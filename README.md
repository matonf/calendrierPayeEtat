# calendrierPayeEtat
Calendrier perpétuel de la paye en fonction publique d'Etat 

Pour avoir dans son calendrier d'ordinateur et de smartphone toutes les dates de paye sans rien faire !
Il suffit de s'abonner à un [flux](https://onfray.info/cache-paye.ics) dans son application ou de cliquer [là directement]("webcal://onfray.info/cache-paye.ics") pour que la magie opère :)
